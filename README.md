# Unity Toast Package

This repository contains a Unity toast package('Releases\ToastPackage.unitypackage') created for the to test Toast notid=fication.

The package provides a Scene with Ui Button that shows a native-style message when clicked:

- **Android** – native `Toast`
- **iOS** – native iOS plugin


## How to Use

1. Open this project in **Unity**.
2. Locate the Scene 'Assets\VaibhavToast\Scenes\TestScene.unity'
3. Play
